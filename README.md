# Electrical Load Billing and Monitoring
## Project Overview
This project simulates electrical load monitoring and electricity billing for a small household or facility.
It allows users to register electrical appliances, calculate energy consumption, and estimate electricity costs.
The program is written in **C++** and runs as a **console-based application**.

## Features
### Appliance Management
- Register electrical appliances
- View all registered appliances

### Energy Calculation
Energy is calculated using:
Energy (kWh) = Power (Watts) × Usage Hours / 1000

### Billing Calculation
Electricity cost is calculated using:
Cost = Energy × Tariff Rate

### File Storage
Appliance data is saved to a text file and loaded automatically when the program restarts.
